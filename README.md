📝 Notes App (React)

A simple, clean, and interactive Notes application built using React (Vite).
It allows users to create, edit, delete, pin, and search notes with a smooth and responsive UI.

---
🚀 Features

- ➕ Create new notes
- ✏️ Edit existing notes
- 🗑️ Delete notes
- 📌 Pin important notes
- 🔍 Real-time search functionality
- 💾 Data saved using localStorage
- 🎨 Modern and minimal UI
- ✨ Smooth animations with Framer Motion
- 📱 Fully responsive design (mobile-friendly)

---
🛠️ Tech Stack

- React (Vite)
- React Router DOM
- Framer Motion
- JavaScript (ES6+)
- CSS (inline styles + global styles)
- localStorage (for data persistence)

---
🎯 Key Highlights

- Fast and lightweight React app
- Clean UI with smooth transitions
- Intuitive note management system
- Beginner-friendly project structure
